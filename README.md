# Pet Wellness Management System

## 🚀 Overview
A full-stack application for managing pet wellness services, including authentication, admin dashboard, and marketplace features.

## 🛠 Tech Stack
- Frontend: React + TypeScript + Tailwind CSS
- Backend: Java (Spring Boot)

## ✨ Features
- User Authentication (Login, OTP, Forgot Password)
- Admin Dashboard
- Marketplace Module
- Responsive UI
